Developing a Geoportal using react js

Title: Developing a maps application using Reactjs and open source tools

Web-maps are becoming more and more popular as a mode of sharing data and information. Open source tools have made the development process easy and more accessible. The completed project is also made available in the github repository

In this tutorial you will learn the following

  • Setting up a new react geoportal project
  • Create a basic wire frame for the geoportal
  • Create different components of your geoportal
  • Setup a redux store for state sharing between components

Setting up a new geoportal project

If you have installed nodejs and npm, you already have the basic setup required to develop the geoportal. You can check the installation by entering the following command in windows command prompt or terminal in case of mac or linux. Incase you have not setup, please follow the link to complete the setup and then proceed.

node -v
npm -v

React based apps a prerequisite of nodejs and npm to be installed beforehand. React app can be setup using command. On successful execution of the command, the output will be shown as below.

npx create-react-app geoportal

Once the application is setup. You may navigate to the geoportal directory and start installing the required packages. The package dependencies required for the development of the geoportal are as follows:

  • react-leaflet (For building interactive maps)
  • react-bootstrap & bootstrap (UI components)
cd geoportal
npm install react-leaflet leaflet
npm install react-bootstrap bootstrap

Once the installation is successful, run the following command to start the application. On successful execution of the command. 

npm start

The next step into start the Open the directory in your favourite code editor (vscode in my case) and navigate to src folder and make a directory to store all your components by typing the following command 

cd src
mkdir components

In order to communicate information between the components, the next step is to create a state management system using redux-toolkit for maintaining the state of the layers that are being checked and unchecked in the layer tree. The map should accordingly display the checked and unchecked layers. Run the following command to install the redux-toolkit.

# NPM
npm install @reduxjs/toolkit

On successful installation of the package. You will be required to setup the broiler plate for implementing the redux store. You will be required to create the following files:

From the project folder in the terminal. Execute the command below to create these files

  • store

    /src/app/store.js 

    cd src
mkdir app
touch store.js

Enter the following code in store

import { configureStore } from '@reduxjs/toolkit';
import counterReducer from '../features/counter/counterSlice';
import layerReducer from '../features/layer/layerSlice';

export const store = configureStore({
  reducer: {
    counter: counterReducer,
    layer:layerReducer
  },
});
  • slice

    /src/features/layerSlice.js

    cd src
mkdir features
touch layerSlice.js

Enter the following code in layerSlice:

The following code creates reducers for the actions performed in the layer tree component. This will help in toggling between the layers.

import { createSlice } from '@reduxjs/toolkit'
import { layer } from "../../config";

const initialState = layer;

const layerSlice = createSlice({
  name: 'layer',
  initialState,
  reducers: {

    setInitialState:(state, action)=>{
        console.log("Test")
        state=action.payload
    },
    changeVisibility:(state,action)=>{

            state[action.payload.id].show = action.payload.show;

    }
  }
});

export const { setInitialstate, changeVisibility} = layerSlice.actions

export const selectLayerDataSet = (state) => state.layer;

export default layerSlice.reducer

Update /src/components/layertree.js to connect it with Redux stores

import React from "react";
import ListGroup from "react-bootstrap/ListGroup";
import Form from "react-bootstrap/Form";
import { useDispatch,useSelector } from "react-redux";
import { increment, setInitialstate, changeVisibility,selectLayerDataSet } from "../features/layer/layerSlice";
import { useEffect, useState } from "react";
const LayerTree = () => {
  
  const dispatch = useDispatch()

  const state = useSelector(selectLayerDataSet);

  const toggleLayer = (id, e) => {
      dispatch(changeVisibility({ ...state, id: id - 1, show: e }));
  };


  //Updating initial redux state on load
  useEffect(() => {
    fetch("./Data.json")
    .then(response => response.json())
    .then(data => console.log(data));

  }, []); 

 return (
    <ListGroup>
      {state.map((layer)=>{
  
        <p>
          {layer.lext}
        </p>
      })}
      <p>Layer Tree</p>
      {state.map((task, index) => (
        // <Layer key={index} task={task} showLayer ={showLayer}/>
        <ListGroup.Item>
        <Form.Check
          type="checkbox"
          id={1}
          label={task.text}
          defaultChecked={task.show}
          onClick={(e) => toggleLayer(1, e.target.checked)}
        />
      </ListGroup.Item>
        ))}

    </ListGroup>
  );
};

export default LayerTree;
